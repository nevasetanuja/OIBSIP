# Task 3 – Temperature Converter

## Project Description
This project is a temperature converter web application that converts temperature values between Celsius, Fahrenheit, and Kelvin.

## Features
- User input field
- Dropdown for unit selection
- Real-time temperature conversion
- Interactive functionality using JavaScript

## Technologies Used
- HTML
- CSS
- JavaScript

## Outcome
Strengthened understanding of JavaScript logic, DOM manipulation, and interactive web development.
